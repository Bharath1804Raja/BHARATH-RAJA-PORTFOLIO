# BHARATH-RAJA-GOVINDASAMY-PORTFOLIO
#  Bharath Raja Govindasamy — Personal Portfolio Website  

Welcome to my personal **developer portfolio**, a modern and responsive website built to showcase my professional profile, projects, and achievements in **Full Stack Web Development**.  

This portfolio acts as my **digital identity**, presenting my skills, career highlights, and online presence — including GitHub, LinkedIn, and resume download options.  
It’s built with **HTML5**, **CSS3 (Tailwind & custom styling)**, and **JavaScript**, designed for both **desktop and mobile** experiences with a clean, minimal, and futuristic look.

---

##  Project Overview  

The **Bharath Raja Portfolio Website** serves as an all-in-one platform for personal branding and professional outreach.  
Visitors can easily:
- Explore my work and featured projects  
- View my coding profiles (GitHub, LinkedIn)  
- Download my latest resume directly  
- Get in touch via the contact form  

It’s fully responsive, smooth in navigation, and optimized for accessibility.

##  Key Features  

###  1. Elegant User Interface  
- Fully responsive layout that adapts to **mobile, tablet, and desktop screens**.  
- Modern design using **TailwindCSS utility classes** with light and dark-friendly colors.  
- Smooth transitions and subtle hover effects for interactive elements.  

###  2. Dynamic Links  
- **GitHub Button:** Instantly opens my GitHub profile — [github.com/Bharath1804Raja](https://github.com/Bharath1804Raja)  
- **LinkedIn Button:** Opens my professional profile — [linkedin.com/in/bharath-raja-govindasamy-244a4b354](https://www.linkedin.com/in/bharath-raja-govindasamy-244a4b354/)  
- **Resume Download:** Allows visitors to download my resume directly to their device (`Bharath_Raja_Resume.pdf`).  

###  3. Contact Integration  
- “Call” and “Email” buttons allow users to directly initiate contact.  
- Contact form is functional — displays a message confirmation (“Message sent ”) when users submit.  
- Form resets automatically for another query submission.  

###  4. Clean Code Separation  
- Organized into **three separate files**:  
  - `index.html` — structure and content  
  - `style.css` — design, colors, responsiveness  
  - `script.js` — interactive functionality (GitHub/LinkedIn links, form validation, resume download)  
- Easy to maintain and expand with additional sections or animations.

##  Design Inspiration  

This design draws inspiration from **modern developer portfolios** that focus on simplicity, accessibility, and futuristic UI.  
The key design principles include:  

- **Glassmorphism:** Transparent layers, glowing borders, and light blur effects.  
- **Consistency:** Unified color palette and consistent typography throughout all sections.  
- **Whitespace Usage:** Clean spacing between sections for a breathable layout.  
- **Futuristic Feel:** Subtle neon glows and gradient overlays.  
- **Accessibility:** Large, readable fonts and high contrast for easy viewing.  

---

##  Responsive Design Behaviour  

The layout is fully responsive and dynamically adjusts:  

| Device Type | Behaviour |
|--------------|-----------|
| **Mobile (<768px)** | Collapsible navigation bar, vertical stacking, auto-adjusted text, tap-friendly buttons. |
| **Tablet (768–1024px)** | Balanced grid layout, images aligned to center, smaller text scaling. |
| **Desktop (>1024px)** | Full-width sections, multiple columns for projects and about content, large hero banner with name and role. |

Uses **CSS Flexbox and Grid** to ensure components always align correctly regardless of screen size.
